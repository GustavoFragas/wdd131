Site Plan: Smart Finance AI

Site Name

Smart Finance AI

I choose this name to show we provide financial advice using artificial intelligence. It look professional.

Site Purpose

A website that provides financial advice to people and companies and resources using artificial intelligence.

Note: this project was thought before. Actually it is in production using other programming languages, but for this college assignment I will build it using only HTML, CSS and JavaScript to attend the course resources and requires.

Scenarios

  • How AI can help me save my business money?
  • Where I find good investments ideas?
  • How me as a small business avoid bankrupt managing my costs? (because I search on internet and it say most small business fail for bad money manage).

Color Schema

About the colors, I think to use Emerald Green and Gold. Because green represent money and growth, and gold or yellow is about prosperity. It make perfect sense to a financial website like this!

  • Emerald Green (#043927) - Used in Headings.
  • Gold (#D4AF37) - Used in borders and accents.

Typography

For typography, I choose Roboto and PT Serif. I use Roboto for the titles because it look more modern. And I use PT Serif on the larger texts and paragraphs, so it make the reading easy for people.
